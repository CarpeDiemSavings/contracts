# Carpediem

**Requirements**

- nodeJS v16.9.1 or later
- hardhat 2.6.4 or later

**Installation**

- yarn

**Run tests**

- yarn covarage
- yarn test

**Deploy contracts**

- set .env and .secret files (see examples)
- yarn deploy


**Verify contracts**

- set .env and .secret files (see examples)
- yarn verify -- 0x.deployed.contracts.address
